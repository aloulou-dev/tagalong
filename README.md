TagAlong is a perosnalized travekl planning application that generates custom itineraries based on a user's desitnation, travel dates, budget, and interests

Features

    - Create personalized day-by-day travel itineraries
    - Discover attraction, restaurants, events, and activities using external APIs
    - Incorporate weather conditions into travel recommendations
    - Save, view, and update trip plans through a database
    - Connect with travelers who share similar destinations, schedules, or interests

Purpose
    TagAlong helps travelers spend less time plannign and more time exploring by providing tailored recommendations and opportunities to connect with others

Technologies
    - Python
    - SQL Database
    - External Travel & Weather APIs

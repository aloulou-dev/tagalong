## README file

TagAlong is a personalized travel planning software that generates custom day-by-day itineraries based on a user’s destination, travel dates, budget, and interests
The platform integrates external APIs to gather information such as attractions, restaurants, and events, while also considering weather conditions
Users can save, view and modify their trips with all itinerary and preference data stored in a database for easy access
TagAlong also helps travelers connect with other users who have similar destinations, schedules, or interests, fostering opportunities to share recommendations, discover activities, and explore together

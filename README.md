## README file

TagAlong is a personalized travel planning application that generates custom itineraries based on a user's destination, travel dates, budget, and interests

- Create personalized day-by-day travel itineraries
- Discover attractions, restaurants, events, and activities using external APIs
- Save, view, and updatye trip plans through a database
- connect with travelers who share similar destinations, schedules, or interests

Use View > Panels > Join with Closest to make one panel.

Click "VS Code" in the top menu bar to open VS Code.
